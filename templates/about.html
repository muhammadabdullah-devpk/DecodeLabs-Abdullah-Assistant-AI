About Abdullah AI

Abdullah Assistant AI is a premium SaaS AI chatbot console that showcases modern Flask backend structure, robust SQLite models, and Google Gemini API streaming integration.

🧠 Technical Architecture

The application follows clean coding patterns separating services and data persistence layers:

  • Flask Blueprints: Compartmentalizes views for landing page, session authentication, AJAX APIs, and streaming response generator layers.
  • Database Persistence: Utilizes SQLAlchemy models for secure user session hashes, pinning states, and messaging transcripts storage.
  • Hybrid Generation Pipeline: Dynamically coordinates standard Google Gemini API client streams with local intent regex fallbacks in case of credentials blockages or connection drops.
